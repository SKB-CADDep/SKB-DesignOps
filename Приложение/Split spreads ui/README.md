## Описание программы
Приложение предназначено для разрезания PDF‑разворотов книг на отдельные страницы с возможностью предпросмотра, смещения линии разреза и поворота страниц. Программа поддерживает разные режимы обработки: линейный порядок (слева‑направо или справа‑налево), а также восстановление книги из типографского спуска (два варианта брошюрного зигзага — вперёд и назад).

Для работы используются библиотеки pypdf (модификация структуры PDF) и PyMuPDF (fitz) для предпросмотра страниц.

## Компилятор
**Шаг 1.** Установить PyInstaller (если не установлен)

**Шаг 2.** Открыть Terminal (alt + F12)

**Шаг 3.** Собрать EXE. Ввести в Terminal: pyinstaller --onefile --windowed --name PDF_Spread_Splitter split_spreads_ui.py

**Шаг 3.** Найти готовый файл в папке dist
