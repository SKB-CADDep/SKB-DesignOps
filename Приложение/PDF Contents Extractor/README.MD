## Описание программы
PDF Contents Extractor — предназначено для извлечения оглавления (закладок) из PDF‑файлов. Программа позволяет выбрать PDF-документ, автоматически получить все закладки в правильном порядке и вывести их в редактируемое текстовое поле.
Приложение поддерживает горячие клавиши (Ctrl+C, Ctrl+V, Ctrl+X, Ctrl+Z, Ctrl+A), контекстное меню (копировать, вставить, выделить всё) и удобное отображение выбранного файла. Подходит для работы с PDF, созданными в Adobe Acrobat.

## Компилятор (как приложение переводить в ЕХЕ)
Шаг 1. Установить PyInstaller (если не установлен)
Шаг 2. Открыть Terminal (alt + F12)
Шаг 3. Собрать EXE. Ввести в Terminal: pyinstaller --onefile --windowed --name PDF_Contents_Extractor PDF_Contents_Extractor.py
Шаг 3. Найти готовый файл в папке dist
